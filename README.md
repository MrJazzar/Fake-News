# 📰 Fake News Detection Project (ML + LSTM Hybrid)

---

## 📌 Overview

This project focuses on **Fake News Detection** using both:

- 🤖 Traditional Machine Learning models  
- 🧠 Deep Learning model (LSTM)

The goal is to classify news articles into:

- 🟢 Real News  
- 🔴 Fake News  

We first build classical ML models for baseline performance, then improve results using an LSTM neural network for sequence learning.
